# HsBlog

一个简介风格的个人博客系统，包含前后台。

## :wrench: 技术栈

- Nuxt3
- Vue3
- Typescript
- UnoCSS
- Naive-UI
- Scss

## 启动

确保安装了依赖项：

```bash
# pnpm
pnpm install
```

## 开发服务器

```bash
pnpm run dev
```

启动开发服务器 `http://localhost:8800`:

## Production

构建用于生产的应用程序：

```bash
# pnpm
pnpm run build
```

本地预览生产构建：

```bash
# pnpm
pnpm run preview
```
